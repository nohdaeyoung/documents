# 메인 페이지 UI 리디자인 제안서

**작성일:** 2026-03-03
**대상 페이지:** d.324.ing (메인 아카이브 목록)
**현황:** 단순 수직 리스트 (`카테고리 태그 | 제목 | 날짜`)

---

## 1. 레퍼런스 조사 요약

### Are.na
- **특징:** 균일 그리드, 호버 시 정보 노출, 극도의 미니멀리즘
- **시사점:** 콘텐츠 자체를 전면에 내세우는 투명한 인터페이스

### Readwise Reader
- **특징:** 밀도 높은 수직 리스트, 읽기 상태 시각화, 강력한 태그 필터
- **시사점:** 리스트가 정보 밀도를 높이는 방향 (제목 아래 부제/요약 추가)

### Notion 데이터베이스
- **특징:** 멀티뷰(리스트/그리드/보드/타임라인), 속성 기반 필터 빌더
- **시사점:** 뷰 전환 + 사이드바 필터가 아카이브 규모 확장에 대비

### 문예지 (n+1, The Paris Review)
- **특징:** 에디토리얼 그리드, 세리프 중심 타이포그래피, 넉넉한 여백
- **시사점:** 현재 사이트 감성(크림·세리프)과 가장 친화적인 레퍼런스

### 학술 아카이브 (arXiv, JSTOR)
- **특징:** 텍스트 밀도 극대화, 초록 미리보기, 패싯 필터
- **시사점:** 내용 미리보기(소개 문장)가 접근성·선택의 질 향상에 기여

## 2. 현재 레이아웃 진단

| 항목 | 현황 | 문제점 |
|------|------|--------|
| 항목 정보 | 태그 + 제목 + 날짜 | 내용 미리보기 없어 클릭 전 콘텐츠 파악 불가 |
| 필터 | 상단 카테고리 탭 | 복수 조건 필터 불가, 연도별 구분 없음 |
| 시각 구조 | 균일한 한 줄 리스트 | 항목 간 위계 없음, 스캔성 낮음 |
| 확장성 | 단순 목록 | 50개 이상 시 스크롤 피로, 탐색 어려움 |

## 3. UI 대안 컨셉 4가지

### 컨셉 A — "에디토리얼 스플릿 리스트" ⭐ 추천

> **레퍼런스:** 문예지 + Readwise | **변경 수준:** 낮음 (점진적 개선)

```
┌──────────────────────────────────────────────────────────┐
│  [필터: 전체 | 철학 | 역사 | 예술 | 문학]               │
├──────────────────────────────────────────────────────────┤
│  2026                                     ───────────── │
│                                                          │
│  철학     봄의 변증법에 대하여              03.15        │
│           강연자의 논지 한두 줄 소개 텍스트.             │
│                                                          │
│  역사     17세기 지도와 권력의 문제         02.28        │
│           짧은 프리뷰. 클릭 전 맥락 파악 가능.           │
│                                                          │
│  2025                                     ───────────── │
│                                                          │
│  예술     영화 언어의 경제학               12.10        │
└──────────────────────────────────────────────────────────┘
```

**변경 사항:**
- 연도 구분 헤더 추가 — 시간적 맥락 명시
- 제목 아래 1-2줄 소개 텍스트 추가 (Firestore `summary` 필드 신설 필요)
- 카테고리를 좌측 고정 칼럼으로 정렬 (스캔성 ↑)
- 현재 타이포그래피·색상 체계 유지

**구현 난이도:** ★★☆☆☆ (2-3일)
**현재 사이트와의 거리:** 매우 가까움

---

### 컨셉 B — "카드 그리드 + 사이드 필터"

> **레퍼런스:** Are.na + Notion 갤러리 | **변경 수준:** 중간 (구조 변화)

```
┌─────────────────────────────────────────────────────────┐
│  [카테고리 ▾]  [연도 ▾]  [정렬 ▾]           ≡  ⊞       │
├────────────┬────────────────────────────────────────────┤
│ 철학  (12) │  ┌──────────┐  ┌──────────┐  ┌──────────┐ │
│ 역사   (8) │  │ 카테고리  │  │ 카테고리  │  │ 카테고리  │ │
│ 예술   (5) │  │ 제목 제목 │  │ 제목 제목 │  │ 제목 제목 │ │
│ 문학   (3) │  │ 날짜      │  │ 날짜      │  │ 날짜      │ │
│            │  └──────────┘  └──────────┘  └──────────┘ │
└────────────┴────────────────────────────────────────────┘
```

**변경 사항:**
- 좌측 카테고리 사이드바 (항목 수 표시)
- 2-3 칼럼 카드 그리드 (카드 상단에 카테고리 색상 바)
- 그리드 ↔ 리스트 뷰 토글
- 드롭다운 필터: 카테고리, 연도, 정렬

**구현 난이도:** ★★★☆☆ (5-7일)
**현재 사이트와의 거리:** 중간

---

### 컨셉 C — "타임라인 뷰"

> **레퍼런스:** 독창적 (강의 아카이브 특성 반영) | **변경 수준:** 높음 (재설계)

```
┌──────────────────────────────────────────────────────────┐
│  2026                                                    │
│                                                          │
│  3월  ●───────────────────────────────────────          │
│       ├ 03.15  봄의 변증법에 대하여       [철학]         │
│       └ 03.08  지도와 권력               [역사]         │
│                                                          │
│  2월  ●───────────────────────────────────────          │
│       └ 02.22  근대성의 기원             [철학]         │
│                                                          │
│  2025 ────────────────────────────────────────[접기]    │
└──────────────────────────────────────────────────────────┘
```

**변경 사항:**
- 월별 타임라인으로 강의 흐름을 내러티브로 표현
- 연/월 단위 접기/펼치기 (Accordion)
- 날짜가 핵심 탐색 축

**구현 난이도:** ★★★★☆ (7-10일)
**현재 사이트와의 거리:** 멀음

---

### 컨셉 D — "제목 중심 풀-타이포그래피 리스트" ⭐ 추천

> **레퍼런스:** 학술·문예지 하이브리드 | **변경 수준:** 낮음 (정제)

```
┌──────────────────────────────────────────────────────────┐
│  철학 · 역사 · 예술 · 문학                   [검색 🔍]   │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  봄의 변증법에 대하여                                    │
│  철학  —  2026. 03. 15                                   │
│                                                          │
│  · · · · · · · · · · · · · · · · · · · · · · · · · ·    │
│                                                          │
│  17세기 지도와 권력의 문제                               │
│  역사  —  2026. 02. 28                                   │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**변경 사항:**
- 제목을 세리프 1.5-2rem으로 키워 전면에 배치
- 카테고리·날짜는 작은 산세리프 서브텍스트로 아래에
- 점선(·) 구분선으로 항목 간 호흡 부여
- 상단 카테고리는 인라인 텍스트 링크 필터
- 태그 색상 박스 제거 → 텍스트 색상으로만 차별화

**구현 난이도:** ★★☆☆☆ (2-3일)
**현재 사이트와의 거리:** 가까움

## 5. 권고사항

### 단기 (이번 주): 컨셉 A 또는 D 선택 후 구현
- 현재 크림·세리프 디자인 언어 유지
- A 선택 시: Firestore에 `summary` 필드 추가, 연도 그룹핑 로직 구현
- D 선택 시: CSS 변경이 주된 작업 (JS 로직 변경 최소)

### 중기 (2-4주): 필드 보강
- 아카이브에 `summary` (1-2줄 소개) 필드 추가
- 관리자 편집 폼에 summary 입력란 추가

### 장기 (필요 시): 컨셉 B 도입
- 아카이브가 50개 이상이 되면 그리드 + 사이드 필터가 탐색 효율 대폭 향상
